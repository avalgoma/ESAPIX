GetComplexDVH(PlanningItem, IEnumerable<Structure>, VolumePresentation, DoseValuePresentation, Double)

The "complex" dvh in this case, is 1. Allows for plan sums to have relative dose, by adding prescriptions to determine a guessed prescribed dose value. (It is up to the user to determine if this is appropriate). 2). Allows for multiple structures to be merged into a single dvh by summing volumes as each dose value
